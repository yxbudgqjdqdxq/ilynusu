# How to add your media

Since the AI Studio environment doesn't automatically pull images from chat or Google Drive seamlessly into the code, you'll need to drop your files right here in this `public/media` folder. 

1. Go to the file menu (left sidebar)
2. Hover over `public/media` and upload your files.
3. Rename them to match the placeholders I've used in the code:
   - `her-video.mp4`
   - `monkey.jpg`
   - `kiss-snap.jpg`
   - `potato-chibi.png`
   - `our-photo-1.jpg`
   - `our-photo-2.jpg`
   - `her-photo-1.jpg`
   - `nasa-1.jpg`

4. For the Scrapbook captions: open `/src/components/Scrapbook.tsx` and change the `caption="something"` to your actual specific memories based on the images!
